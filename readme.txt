This plugin allows developers to store and access a range of client variables

== Description ==

This is a developer plugin that allows for content that might normally be hard-coded into a custom build (such as links to social media, phone numbers, and opening hours) to be accessible from WordPress.

Developed by [James Robinson](https://jmr.codes).

== Installation ==

1. Download the [latest release](https://github.com/jmrcodes/jmr-variables/releases/latest)
2. Unzip the plugin and copy the `jmr-require-login` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
